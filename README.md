# Automation Playground

A collection of workflow automations built using n8n, OpenAI, Google Workspace, and other productivity tools.

## Technologies

- n8n
- OpenAI API
- Google Drive API
- Gmail API
- Google Sheets
- Webhooks

## Categories

- Google Workspace Automation
- AI Automation
- Productivity
- HR Automation
- Utilities

## Featured Workflows

### Google Drive File Organizer

Automatically organizes uploaded files into predefined folders.

### AI Resume Parser

Extracts structured resume data using OpenAI.

### Cover Letter Generator

Generates ATS-friendly cover letters based on resume content.

### Gmail Automation

Automatically processes incoming emails and routes attachments.
